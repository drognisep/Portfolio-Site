DougTube - My LAN version of YouTube

Have you ever experienced the pain of trying to stream your videos to different devices throughout your home? Me too, that's why I decided to build a LAN version of YouTube utilizing the new <video> tag that comes with HTML5.

It operates using a WAMP server hosting the service behind a secured wireless router and a PHP login prompt. That way I can prevent anyone from viewing my digital copies.

I plan on creating a Java dashboard that can automate user and content management so I don't have to manually update configuration files to make this work.
