Cash to close calculator

Purpose

To create an easy to use utility for one of America's largest lenders that simplifies the calculation of cash needed to close a mortgage.

Features

  • Utilizes a mainframe access type library to pull relevant data - reducing the need for manual processing and increasing accuracy
  • Posts calculated figures to an Excel based Loan Estimate generation utility
  • Provides clear, concise labels to the user to help make the tool's use more intuitive.
